# Project overview
This is a Udacity Blockchain Capstone project to build a real estate token. A "Monster House Token" was used as a joke between friends.

# Evaluation parameters
## Versions
- Truffle v5.3.2 (core: 5.3.2)
- Solidity - 0.5.17 (solc-js)
- Node v10.18.0
- Web3.js v1.3.5

## How to test

 - Run `npm install` to install dependencies
 - Install truffle if not already installed (`npm install -g truffle`)
 - In a terminal, run `ganache-cli`
 - In another terminal, run `truffle test`
